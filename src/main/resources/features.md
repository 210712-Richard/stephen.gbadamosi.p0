# Online Banking

## User Stories

** Customer Access **
* Register
* Create Account
* Deposit/withdraw into/from an account

** Teller Access **

# GameShop

## User Stories

** Customer Access **
* Register for online services
* Rent or buy video games
* Earn points
* Use points
* Request a title


** Clerk/Admin Access **
* Create new clerk account
* Manage customer account
* Manage inventory
* Approve title requests
